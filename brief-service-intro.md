Summary:

### 앱 이름 및 핵심 컨셉

- 앱 이름을 **"그리니**로 결정
- 도메인은 [greeny.my](http://greeny.my) 또는 [greeny.ai.kr](http://greeny.ai.kr)로 등록 예정 (연간 15,000~16,500원)
- 핵심 차별점: **인격화된 식물을 통한 교감**
- 피크민(Pikmin)과 유사한 컨셉으로 식물 캐릭터와 애착 형성이 중요
- 사용자가 앱에 들어가자마자 자신의 텃밭/화분을 보여주는 것이 핵심

### 경쟁사 분석 결과

- **식물 관리 앱들** (BloomBox, Planta 등): 전문 사용자 대상, UI/UX가 복잡
- **AI 일기 앱들** (Sims, Mindy, 하루콩 등): 인격화된 AI와 대화하는 기능이 있지만 누적된 대화가 인격을 만드는 앱은 없음
- **Finch**: 습관 형성 앱으로 귀여운 캐릭터와 상호작용하지만 직접적인 대화는 부족
- 현재 시장에는 명확한 리더가 없음 - 기회이자 도전 과제

### 주요 기능 및 UX 설계

- **메인 화면**: 텃밭/화분 뷰가 처음 화면, 식물들이 귀엽게 표시됨
- **식물 캐릭터**: 3개 슬롯에 식물 배치, 각 식물은 고유한 인격을 가짐
- **AI 대화 시스템**:
    - 식물이 먼저 말을 걸어야 함 (손톱 메시지)
    - 대화 내용이 자동으로 일기/기록으로 변환
    - 누적된 대화가 식물의 인격을 형성
- **게이미피케이션**: 젤다의 야생의 숨결처럼 넛징(nudging) 기법 활용, 튜토리얼 없이 자연스러운 유도

### 온보딩 및 초기 설정

- 처음에 권한 요청하지 말고 식물 등록부터 시작
- 식물 등록 → 텃밭에 식물 표시 → 날씨/위치 권한 요청 순서로 넛징
- AI로 식물 종류 자동 인식 기능
- 물주기 리마인더를 종별 주기에 맞춰 자동 설정

### 네비게이션 구조

- 5개 탭 구조 제안: 피드 - (작은 탭) - **내 정원**(가운데, 크게) - (작은 탭) - 설정
- 커뮤니티는 별도로 만들지 않고 피드로 통합 (헤비 유저 100명 이상 필요)
- 식물 도감 포함
- 채팅은 별도 탭 없이 각 식물 화면에서 접근

### 카카오톡 오픈채팅방 분석

- 식집사 오픈톡방 780명 분석 결과
- 가장 많은 주제: 식물 관리 상담, 병해충 문제
- 질의응답이 활발하게 이루어짐
- 많은 사용자들이 GPT를 활용하고 있음

### 페인 포인트 및 필수 기능

- **물주기 알람**: 가장 기본적이고 중요한 기능
- **증상 진단**: AI가 1차 진단 후 커뮤니티에 자동 게시, 전문가/사용자가 댓글로 조언
- 날씨 정보 자동 로드 및 관리 조언
